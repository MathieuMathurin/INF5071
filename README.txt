Tchernobyl Jumper

Il y a deux executables pour pouvoir jouer au jeu directement 
jeuRunnable1Joueurs et jeuRunnable2Joueurs

Sinon on peut jouer directement dans Unity
Pour se faire:
1. ouvrir Unity
2. ouvrir le projet "scene_tp1" dans Assets
3. Clicker sur play en haut pour jouer
4. Supprimer le Character 2 pour jouer Ã  un seul joueur
5. On peut le remettre facilement, il se trouve ici: Assets\Standard Assets\2D\Prefabs\Character 2.prefab
